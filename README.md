# HU Inventory Management System

Haramaya University – College of Computing and Informatics

**Live Demo:** https://firomsa51.github.io/ipproject.github.io/

## Features
- Stock item catalog (add, edit, delete)
- Supplier directory management
- Stock movement logging (inflow/outflow/damage)
- Store categories
- Asset valuation dashboard
- Low stock alerts
- User authentication (login/signup)
